# 프로젝트 범위(Project Scope)

## 프로젝트 목적(Objective)

SauceDemo 웹 서비스의 핵심 사용자 기능을 대상으로
품질 검증(Quality Assurance)을 수행한다.

수동 테스트(Manual Testing),
자동화 테스트(Test Automation),
CI/CD 기반 테스트 환경 구축을 목표로 한다.

---

## 테스트 범위(In Scope)

### 1. Login
사용자 인증 및 로그인 기능 검증

### 2. Product Inventory
상품 목록 조회 기능 검증

### 3. Product Sorting
상품 정렬 기능 검증

### 4. Cart
장바구니 추가/삭제 기능 검증

### 5. Checkout
주문 정보 입력 및 구매 프로세스 검증
